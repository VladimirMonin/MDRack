# v0.4 offline release-candidate packet

Status: ready for independent documentation review; not published.

The committed implementation base is Git revision
`7859c55a096c2acc20d91181a3df459d8181bc89`. Verification applies to that base
plus only the uncommitted, content-addressed files listed under
`candidate_snapshot.build_inputs` in the JSON companion. The candidate is
materialized before the artifact build; `publication_outputs`, including the
artifact manifest, packet, validator, release-only tests, architecture links, and
ADRs, are generated or validated afterward and are never copied into candidate
bytes. This removes the root-sdist self-reference: artifact hashes cannot change
when a publication output records those hashes. It is not an exact-HEAD
verification claim, and no commit, tag, push, package upload, or release
publication is claimed.

The machine-readable companion is
[`v0.4-release-packet.json`](v0.4-release-packet.json). Public ownership,
version, deprecation, removal, and installed-test mappings are in the
[`v0.4 public-surface ledger`](../compatibility/v0.4-public-surface-ledger.json).

## Quality and performance boundaries

The local provider-free quality command processed 170 cases and 120 units with
`lexical-token-overlap-v1`. Aggregate Recall@5 was `0.191176470588`, Recall@10
`0.320588235294`, MRR `0.15858076564`, nDCG `0.166446628192`, duplicate rate
`0.0`, and interval-hit@K `0.628571428571` over 70 temporal cases. These are
frozen orchestration-baseline values, not semantic-provider quality.

The SQLite envelope is local-components evidence only. The checked report measures
1k/10k units at 384/768/1024 dimensions. The 10k×1024 cell exceeds the chosen
latency warning threshold and is `bounded/slow`; 50k and 100k cells are unmeasured.
No PostgreSQL, ANN, or backend cutover follows automatically.

## Privacy and publication review

The packet contains public synthetic IDs, aggregate metrics, hashes, repository
paths and Git revisions only. It contains no source text, raw query text, private
path, endpoint, provider body, vector, credential, database, or user corpus.

Explicit non-claims:

- Windows and Python 3.12 were not executed here.
- No live LM Studio/OpenRouter request was made.
- No real source, private vault, OCR, Whisper, VLM, visual, or acoustic quality was exercised.
- Frame-caption retrieval is not enabled as a production default by this packet.
- No final `1.0` stability is declared.
- No package, tag, push, deployment, or release publication occurred.
