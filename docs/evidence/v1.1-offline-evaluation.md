## Ablations

Each repeat records all configured families:

- timed chunking: compact, balanced, large;
- frames: transcript-only graph, speech-first, balanced;
- metadata: body-only, title projection, aliases, selected metadata with store-only exclusion;
- hybrid: lexical-only (1.0/0.0), static-text semantic-only (0.0/1.0), equal (0.5/0.5) and configured (0.4/0.6) weights. One runtime-contract profile table drives four executions over the same 30 frozen hybrid cases in each repeat; every cell persists the same metric-object schema rather than a label or placeholder.

The vector fixture is generated from public text only and does not read judgments. It proves deterministic retrieval orchestration, not provider or universal semantic quality.
