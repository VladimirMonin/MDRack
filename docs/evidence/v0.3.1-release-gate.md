### Reproduction procedure

Run from the repository root. The harness uses only standard-library modules and
MDRack's migration/store classes; it inserts deterministic synthetic rows directly
into a disposable candidate schema so fixture construction is excluded from the
search-process RSS. Repeat a cell by changing `N` and `DIM`.

```bash
N=10000 DIM=384 uv run python -c '
import json, os, sqlite3, subprocess, sys, tempfile
from pathlib import Path
from mdrack.storage.sqlite.connection import get_connection
from mdrack.storage.sqlite.migrations import apply_candidate_migrations, get_migrations_dir
n, dim = int(os.environ["N"]), int(os.environ["DIM"])
with tempfile.TemporaryDirectory(prefix="mdrack-v031-offline-") as temp:
    db = Path(temp) / "bench.db"
    conn = get_connection(db)
    apply_candidate_migrations(conn, get_migrations_dir())
    now = "2026-07-18T00:00:00+00:00"
    conn.execute("INSERT INTO core_resources VALUES(?,?,?,?,?,?,?,?,?,?,?)", ("r","synthetic","application/x.synthetic","offline-benchmark","synthetic","{}","sha256:" + "0" * 64,None,None,"{}",now))
    conn.execute("INSERT INTO core_representations VALUES(?,?,?,?,?,?,?,?,?,?)", ("p","r","whole_resource","text",None,None,None,None,None,"{}"))
    conn.execute("INSERT INTO core_embedding_spaces VALUES(?,?,?,?,?)", ("s",dim,"cosine","offline-benchmark-v1","{}"))
    blob = json.dumps((1.0,) + (0.0,) * (dim - 1), separators=(",", ":")).encode()
    for start in range(0, n, 1000):
        stop = min(n, start + 1000)
        conn.executemany("INSERT INTO core_search_units VALUES(?,?,?,?,?,?,?,?,?,?,?,?)", ((f"u-{i:06d}","r","p","whole_resource","text",None,"synthetic","{}",i,None,None,"{}") for i in range(start, stop)))
        conn.executemany("INSERT INTO core_unit_embeddings VALUES(?,?,?,?)", ((f"u-{i:06d}","s",blob,now) for i in range(start, stop)))
    conn.commit(); conn.close()
    child = """
import json, resource, socket, sys, time
from mdrack.adapters.sqlite.resource_store import SQLiteResourceStore
from mdrack.storage.sqlite.connection import get_connection
from mdrack_core.domain import SearchScope, VectorBranch
attempts = 0
def blocked(*args, **kwargs):
    global attempts; attempts += 1; raise AssertionError("network disabled")
socket.create_connection = blocked
dim, db = int(sys.argv[1]), sys.argv[2]
conn = get_connection(db); store = SQLiteResourceStore(conn)
branch = VectorBranch("bench", "s", (1.0,) + (0.0,) * (dim - 1), 1.0, 10)
store.search_vector(branch, scope=SearchScope())
times = []
for _ in range(3):
    started = time.perf_counter(); result = store.search_vector(branch, scope=SearchScope()); times.append(time.perf_counter() - started)
assert len(result) == 10
print(json.dumps({"latency_seconds": times, "median_seconds": sorted(times)[1], "peak_rss_kib": resource.getrusage(resource.RUSAGE_SELF).ru_maxrss, "network_attempts": attempts}))
"""
    subprocess.run([sys.executable, "-c", child, str(dim), str(db)], check=True)
'
```

## Separately gated semantic benchmark

No live semantic benchmark was run. It requires a new explicit approval for LM
Studio model execution and a bounded runtime/call budget. Inputs must be an
immutable public or synthetic corpus without PII, immutable relevance judgments,
a pinned model/profile/fingerprint, and frozen lexical/semantic/equal-weight/
configured-weight baselines.

Success criteria are complete Recall@5/10, MRR@10, nDCG@10, latency p50/p95,
index time and vector count, with privacy-safe immutable evidence. The
non-destructive procedure is: create a new disposable database; verify the model
runtime starts empty; execute each frozen baseline once; capture sanitized metrics
and digests; unload; verify the runtime is empty; delete only the disposable
database. Active user stores, active generations, private corpora, paid APIs and
retrying a failed live call are forbidden without separate approval.
