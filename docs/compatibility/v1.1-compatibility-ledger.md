Status: B0 baseline inventory
Baseline: `989aaaeaf029d7bdd2d9704ae578c9a19efe0f94`

This ledger distinguishes implemented foundations from missing application
behavior. `present` means present at the B0 baseline; it does not mean final 1.0
or 1.1 release acceptance. `missing_at_b0` is an assigned future stage, not a
request to alter a frozen package owner in B0.

## Preserved public compatibility

- `mdrack.public_api.__all__` remains the v0.3 compatibility surface until an
  owning 1.1 stage adds and tests an additive export.
- `PreparedResourceCatalog` remains published from
  `mdrack.application.resource_catalog`; B0 does not widen the frozen
  `mdrack.public_api.__all__`.
- Existing `resource`, `resources`, `similar`, `facets`, image, Markdown,
  retrieval, status, doctor, and model CLI behavior remains in place.
- Manifest version 1 import remains strict and closed. Export is not present at
  B0 and is owned by U1.
- Legacy file/section SQLite IDs remain a documented compatibility asymmetry;
  no new 1.1 contract may copy it.

## Explicit non-claims

B0 does not declare final 1.0, final 1.1, a branch/tag, Windows or Python 3.12
execution, a live provider, paid/network work, private-source quality, schema
migration, destructive cleanup, package publication, or release publication.
