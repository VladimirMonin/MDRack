## Evidence boundary

The acceptance test builds wheels from the three local package projects, creates an
isolated Python 3.11 environment, installs only those wheels with `--no-index`, and
runs the fixture with `PYTHONPATH` removed. This proves an installed-package consumer
can exercise the frozen contracts offline. It does not prove WVM integration, live
providers, Windows behavior, or a deployed external host.
